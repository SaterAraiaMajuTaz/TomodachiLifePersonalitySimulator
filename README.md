index.htmlを開くとトモコレの性格をシミュレーションできる。

仕組み
トモコレの性格は、このように決まる。

「行動は」と「言葉は」のパラメータの合計が5以下のとき、
性格分布図の一番左側になる。

「行動は」と「言葉は」のパラメータの合計が13以上のとき、
性格分布図の一番右側になる。


「行動は」が1番左で「言葉は」が5番目〜7番目

「行動は」が2番目で「言葉は」が4番目〜6番目

「行動は」が3番目で「言葉は」が3番目〜5番目

「行動は」が４番目で「言葉は」が2番目〜4番目

「行動は」が５番目で「言葉は」が一番左〜3番目

「行動は」が6番目で、「言葉は」が一番左と3番目

「行動は」が7番目で、「言葉は」が一番左か2番目

「行動は」が8番目で、「言葉は」が一番左

これらに当てはまれば、性格分布図の左から2行目のところになる。


「行動は」が1番左で「言葉は」が8番目

「行動は」が2番目で「言葉は」が7番目か8番目

「行動は」が3番目で「言葉は」が6番目〜8番目

「行動は」が４番目で「言葉は」が５番目〜8番目

「行動は」が５番目で「言葉は」が５番目〜7番目

「行動は」が6番目で、「言葉は」が4番目と6番目

「行動は」が7番目で、「言葉は」が3番目〜5番目

「行動は」が8番目で、「言葉は」が2番目〜4番目

これらに当てはまれば、性格分布図の左から3行目のところになる。



「表情は」と「考え方は」のパラメータの合計が5以下のとき、
性格分布図の一番下側になる。

「表情は」と「考え方は」のパラメータの合計が13以上のとき、
性格分布図の一番上側になる。


「表情は」が1番左で「考え方は」が5番目〜7番目

「表情は」が2番目で「考え方は」が4番目〜6番目

「表情は」が3番目で「考え方は」が3番目〜5番目

「表情は」が４番目で「考え方は」が2番目〜4番目

「表情は」が５番目で「考え方は」が一番左〜3番目

「表情は」が6番目で、「考え方は」が一番左と3番目

「表情は」が7番目で、「考え方は」が一番左か2番目

「表情は」が8番目で、「考え方は」が一番左

これらに当てはまれば、性格分布図の下から2列目のところになる。


「表情は」が1番左で「考え方は」が8番目

「表情は」が2番目で「考え方は」が7番目か8番目

「表情は」が3番目で「考え方は」が6番目〜8番目

「表情は」が４番目で「考え方は」が５番目〜8番目

「表情は」が５番目で「考え方は」が５番目〜7番目

「表情は」が6番目で、「考え方は」が4番目と6番目

「表情は」が7番目で、「考え方は」が3番目〜5番目

「表情は」が8番目で、「考え方は」が2番目〜4番目

これらに当てはまれば、性格分布図の下から3列目のところになる。

「はっきりいって」は、性格には影響しない。

例:

8,1,1,8,1→ ナゴミ系まったり型

1,8,8,1,8→ドライ系テキパキ型
